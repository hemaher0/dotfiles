# Bundled Fonts

This directory contains terminal fonts that can be installed without network access.

The install scripts copy files from this directory only. They do not clone repositories or download release archives.

## Font Set

| Directory | Font family |
| --- | --- |
| `RedHatMono/` | Red Hat Mono |
| `D2Coding/` | D2CodingLigature Nerd Font |

The mono D2Coding font is available for terminal environments that need a monospace Nerd Font.
